## Design Tokens（结构化布局参数）

所有主题通过 `_theme()` 工厂函数自动包含以下 token（由 `_LAYOUT_DEFAULTS` 兜底）：

| Token | 默认值 | 用途 |
|-------|--------|------|
| `slide_w` | 13.333" | 幻灯片宽度（16:9 默认），4:3 时自动设为 10.0" |
| `slide_h` | 7.5" | 幻灯片高度（固定） |
| `cover_title_size` | 36 pt | 封面/标题页大字号 |
| `page_title_size` | 16 pt | 内容页标题字号（投行级标准） |
| `chart_subtitle_size` | 11 pt | 图表副标题字号 |
| `footer_size` | 8 pt | 页脚来源/页码字号 |
| `header_y` | 0.25" | 页眉标题 y 坐标 |
| `divider_y` | 0.80" | 标题下分隔线 y 坐标 |
| `divider_h` | 0.015" | 分隔线高度 |
| `content_y` | 1.00" | 内容区起始 y 坐标 |
| `content_h` | 5.80" | 内容区高度 |
| `content_w` | 自动派生 | 内容区宽度 = `slide_w - 2 * margin`（16:9 下为 12.133"） |
| `footer_y` | 7.10" | 页脚 y 坐标 |
| `table_header_bg` | 跟随 primary | 表头背景色 |
| `table_zebra_even` | `F8FAFC` | 表格偶数行背景 |
| `table_zebra_odd` | `FFFFFF` | 表格奇数行背景 |
| `chart_default_scheme` | 跟随主题名 | 图表自动配色方案 |

**比例预设**：`resolve_theme(theme, aspect_ratio="4:3")` 会自动将 `slide_w` 设为 10.0"，并重新计算 `content_w`。所有布局均使用 `theme["slide_w"]` 而非硬编码值。

**重要**：图表配色现在自动关联主题。使用 `jp_finance` 主题时，图表默认使用 `jp_finance` 配色，无需手动指定 `style_config`。

## 设计原则

### 从顶级投行研报提炼的金融报告配色规律

1. **主色调不超过 2 种**：一个中性深色（标题/边框），一个品牌亮色（折线/强调）
2. **金色/琥珀作为第三色**：用于参考线、平均线、KPI 强调数字——几乎所有顶级金融报告都用这个技巧
3. **浅色背景要极浅**：内容页背景不是纯白，而是极浅的品牌色调（如 `#F4F6FA`，极浅蓝灰）
4. **正负色要和品牌色区分**：不能让绿色和主色相撞，不能让红色和强调色相撞
5. **图表系列色的排列逻辑**：第1色给面积最大的柱/面积图（用中性色），第2色给折线（用饱和度高的色），后续色交替深浅

## 主题一：jp_finance（金融/银行）

**设计意图**：顶级投行同款风格。深海军蓝 + 品牌 Teal + 金色。稳重、专业、信赖感。
**参考**：国际投行与头部券商公开年报

### 颜色规格

| 变量 | 色值 | 预览说明 |
|------|------|---------|
| `primary` | `1B3D6E` | 深海军蓝——标题栏、表头、KPI数字 |
| `secondary` | `00A5BD` | 品牌 Teal——装饰线、折线图主线 |
| `accent` | `C9A84C` | 金琥珀——强调线、平均线、KPI高亮 |
| `bg_dark` | `0F2340` | 极深海军——标题页/结论页背景 |
| `bg_light` | `F4F6FA` | 极浅蓝灰——内容页背景（接近白色但有温度） |
| `card_bg` | `FFFFFF` | 纯白——卡片/面板背景 |
| `text_dark` | `1A2744` | 深蓝黑——浅底正文 |
| `text_light` | `FFFFFF` | 纯白——深底文字 |
| `text_muted` | `6B7C93` | 蓝灰——脚注、数据来源、次要说明 |
| `border` | `C8D6E5` | 浅蓝灰——卡片边框、分割线 |
| `positive` | `0B7B3E` | 森林绿——正收益、上涨 |
| `negative` | `C0392B` | 标准红——负收益、下跌 |

### 排版规格

| 变量 | 值 |
|------|-----|
| `header_font` | 微软雅黑 |
| `body_font` | 微软雅黑 |
| `title_size` | 28 pt |
| `subtitle_size` | 18 pt |
| `body_size` | 12 pt |
| `caption_size` | 9 pt |

### 图表系列配色（8色序列）

```
色1  A8D5DC  —— 浅teal，柱状图/面积图（面积最大，用浅色避免压迫感）
色2  1B3D6E  —— 深海军蓝，主折线（对比最强）
色3  C9A84C  —— 金琥珀，参考线/第二折线
色4  4A8FB8  —— 中蓝，第三系列
色5  7FBFCF  —— 中浅teal，第四系列
色6  A07840  —— 深琥珀，第五系列
色7  8FA8C0  —— 蓝灰，第六系列
色8  505868  —— 深灰，负向/中性系列
```

### 热力图配色

- **红蓝发散**（用于通胀、利率偏离）：`C0392B` → `F5CBA7` → `FFFFFF` → `AED6F1` → `1B3D6E`
- **绿红发散**（用于PMI扩张/收缩）：`0B7B3E` → `A9DFBF` → `FFFFFF` → `F5B7B1` → `C0392B`

### 章节色标签（左侧竖条）

| 章节 | 色值 |
|------|------|
| 宏观经济 | `00A5BD`（secondary） |
| 权益资产 | `C9A84C`（accent） |
| 固定收益 | `1B3D6E`（primary） |
| 另类资产 | `4A8FB8` |
| 投资原则 | `0B7B3E` |

## 颜色使用优先级规则（适用所有主题）

1. **双轴图表**：左轴系列用 `色1`（柱/面积），右轴系列用 `色2`（折线）
2. **单轴多系列**：按 `色1→色3→色5→色7`（奇数色偏浅）或 `色2→色4→色6→色8`（偶数色偏深）交替
3. **参考线/均值线**：固定用 `accent` 色 + 虚线样式
4. **预测区域**：主系列改为虚线，颜色不变，透明度降低（视觉上用深色 dash 表示不确定性）
5. **零线**：固定用 `text_dark` 色，加粗（1.5pt）
6. **正/负方向着色**：柱状图正值 `positive`，负值 `negative`（适用于瀑布图、差值图）
